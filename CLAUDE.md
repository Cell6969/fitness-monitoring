# AGENTS.md

## Project

Healthy Diet Website

## Objective

Build a clean, modern, responsive healthy diet website using Astro.

The project is a static website only.

No backend.

No authentication.

No API.

No database.

Focus on beautiful UI, maintainability, accessibility, and performance.

---

## Tech Stack

Framework:

* Astro

Styling:

* css or astro

Icons:

* Lucide Astro

Deployment:

* Vercel / Netlify

---

## Code Style

* Use TypeScript.
* Use Astro Components.
* Keep components reusable.
* Avoid duplicated code.
* Use semantic HTML.
* Follow accessibility best practices.
* Prefer composition over large files.

## Animations

Use subtle transitions.

Avoid excessive motion.

Hover:

* translateY(-4px)
* shadow-lg

Duration:

200ms

---

## Performance Rules

* Optimize images.
* Lazy load non-critical assets.
* Avoid unnecessary JavaScript.
* Prefer Astro Islands only when interaction is needed.

---

## SEO

Every page should include:

* Title
* Description
* Open Graph Tags
* Twitter Cards

---

## Accessibility

Use:

* semantic HTML
* aria-label
* alt text
* keyboard navigation

---

## Coding Principles

* Keep files small.
* Single Responsibility Principle.
* Reusable Components.
* Avoid inline styles.
* Use Tailwind utilities.

## Future Ready

Design components so future features can be added easily:

* Dark Mode
* CMS
* Search
* Weekly Meal Plan
* Calories Calculator
* BMI Calculator

Do not implement these features yet, only keep the architecture extensible.
